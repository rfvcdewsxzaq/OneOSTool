# 星越L 2021-2023款 车机助手

针对吉利星越L车型（2021-2023款）开发的车机功能增强工具，使用Antigravity （Gemini 3.0）生成代码。

![NaviTool](https://github.com/user-attachments/assets/515f9cdf-f55e-4948-bce8-19bd29eaa551)


## 主要功能

1. **系统主题模式同步至高德地图车机版**  
   自动将车机系统的深色/浅色主题设置同步至高德地图车机版，并实现监控高德运行状态，确保主题成功切换。

2. **强制开启车机日夜切换主题（光线感应）**  
   启用或强制车机系统根据环境光线自动切换日夜主题模式（2024及2025款可以尝试使用）。

3. **方控按键支持第三方音乐播放器**  
   允许使用方控按键操作第三方音乐播放器（冬瓜、汽水），需要仪表及HUD显示歌词请安装MConfig+。

4. **微信按键自定义启动应用**  
   实现短按、长按微信按键启动其他应用，后期可能增加新功能。

5. **车辆启动后自动启动应用**  
   设置发动机启动后自动运行指定的应用程序。

## 使用说明

- 工具自启动需要授予无障碍权限（使用应用管家授予权限）。
- 编译需要使用Android Studio，并提取ecarx.jar和ecarx.adaptapi.jar，经dex2jar处理后放入/app/libs/中。
