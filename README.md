# 星越L 2021-2023款 银河OS工具箱

针对吉利星越L车型（2021-2023款）开发的银河OS功能增强工具，完全使用Antigravity （Gemini 3.0）生成代码。

![OneOSTool](https://github.com/rfvcdewsxzaq/OneOSTool/blob/main/assets/main.png)


## 主要功能

1. **系统主题模式同步至高德地图车机版**  
   自动将车机系统的深色/浅色主题设置同步至高德地图车机版，并实现监控高德运行状态，确保主题成功切换。

2. **车机主题模式切换**  
   除系统主题设置浅色深思以及根据日出日落时间切换外增加根据光线感应自动切换身前模式。

3. **方控按键支持第三方音乐播放器**  
   允许使用方控按键操作第三方音乐播放器（QQ车机版、冬瓜、汽水等）。

4. **车机亮度锁定功能**   
   强制锁定车机亮度，修复银河OS2.3.0版本360影像非正常退出后亮度无法恢复(俗称“闪光弹”)的BUG。
     
5. **微信按键自定义功能**  
   实现短按、长按微信按键启动其他应用，后期可能增加新功能。

6. **车辆启动后自动启动应用**  
   设置发动机启动后自动运行指定的应用程序。

7. **增加自带QQ音乐媒体信息广播功能**   
   获取车机自带QQ音乐媒体信息，模拟发送标准媒体广播，配合Mconfig+可实现HUD显示歌曲信息

## 使用说明

- git clone https://github.com/rfvcdewsxzaq/OneOSTool.git。
- 将ecarx.jar和ecarx.adaptapi.jar，经dex2jar处理后放入/app/libs/中。
- Android Studio编译调试.
- 工具自启动需要授予无障碍权限（使用应用管家授予权限）  
   *注意：出于安全考虑ecarx.jar和ecarx.adaptapi.jar不随源码提供请自行从车机提取*
